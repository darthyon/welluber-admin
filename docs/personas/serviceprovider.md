# Persona: Service Provider Admin

> **Route Group:** `(serviceprovider)`
> **URL Pattern:** `/[spSlug]/dashboard`, etc.

---

## Role Description

Admin staff at a registered service provider (e.g., gym, clinic, wellness center). Manages their branches, voucher redemptions, and settlement information.

## Capabilities

### Subset of Host
- [x] View SP-specific dashboard
- [x] Manage own branches
- [x] View voucher redemptions
- [x] View settlement reports
- [x] SP-level profile/settings

### Cannot
- [ ] View organizations or their employees
- [ ] Create/manage policies
- [ ] Access system settings
- [ ] View other providers
